# Business Name Generator AI Pro

Developed by Abdulrahman Mostafa

## Requirements

- Python 3.11+
- pip

## Installation

Install packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:8080
```

## Features

- AI Business Name Generator
- Brand Kit Generator
- Logo Prompt Generator
- Brand Colors
- Brand Style Guide
- Save Projects
- Export Reports
- Responsive Design
